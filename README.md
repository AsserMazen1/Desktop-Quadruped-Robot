# Desktop-Quadruped-Robot
(This repositry is not up to date)
A Robot Inspired by the Micro Sesame Project: https://github.com/dorianborian/sesame-robot-micro

But with the aim of making a version that has autonomus capabalities such as robots like Cosmo or Vector (by Anki)

# Parts:
## Movement:

Motor (8x): DM-S0020 (4.8v-6.v + 35mA) + (Connector Wire Length: 110mm)

Controller (1x): ESP32-S3

IMU: GY-9250

TOF: TOF200C-VL53L0X

## Main:
Communicator (1x): Luckfox pico mini linux rv1103

Screan (1x): ER-OLEDMO_49

Camera (1x): OV2640

Speaker (1x): MAX98357A I2S Audio Amplifier Module

Mic (1x): INMP441 I2S Microphone Module
